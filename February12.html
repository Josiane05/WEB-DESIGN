Introduction to CSS

CSS stands for cascading Style Sheet. it is a Scripting language which deals with improving the presentation of our web content:

Advantages of CSS

  • It saves time while programming
  • It saves storage space
  • It reduces the download size
  • It improves the appearance of web content

How to insert CSS codes in HTML

We normally have only 3 main ways of inserting codes in HTML:

  1. Inline CSS: We directly insert CSS codes using Style key word.
  2. Internal CSS: We only put CSS codes in the Heading
  3. External CSS: we create separate file and we put all CSS codes in this file. then we call this file in HTML

Types of Selectors

We usually have 4 types of selectors:

  • Element Selector
  • ID Selector
  • Class Selector
  • Pseudo Selector
